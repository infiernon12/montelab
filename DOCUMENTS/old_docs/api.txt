Endpoint /license/check должен требовать заголовок X-Signature с HMAC-SHA256 подписью тела запроса

Валидация на стороне сервера:

Проверка свежести timestamp (±2-3 минуты)

Проверка уникальности nonce (предотвращение replay-атак)

Верификация HMAC подписи

Rate Limiting: Не более 60 запросов в минуту с одного IP

Все даты возвращаются в формате ISO 8601 (UTC)

Ответы всегда содержат signature для проверки подлинности на клиенте
